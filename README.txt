CMSC 12200 Winter 2018
Data driven chess engine (Ginormous Anteaters)

Uses open source GPL and MIT licenses of Sunfish and chessboard.js,
respectively. (see subdirs for licenses). 

Run with "python3 manage.py runserver" to start server on stats page.
Run chess game by putting "/chessboard" in url.
